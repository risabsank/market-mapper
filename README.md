# Market Mapper

Initial scaffold for the Market Mapper project described in `scope.md`.

This repository is currently organized around five main areas:

- `docs/` for implementation planning and project docs
- `src/market_mapper/` for backend, workflow, agent, schema, and sandbox code
- `frontend/` for the dashboard and session chat UI
- `tests/` for unit, integration, and fixture-based tests
- root docs such as `AGENTS.md` and `scope.md`

The current structure is intentionally light. It defines clear module boundaries without pretending the system is already implemented.
